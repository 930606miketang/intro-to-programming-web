Attributes

Attributes are special words inside the opening tag of an element that provide extra information.

Example: The Anchor Tag

The tag is used to create hyperlinks. The most common attribute is href, which indicates the destination URL.

<a href="https://www.example.com">Go to EXAMPLE site</a>
