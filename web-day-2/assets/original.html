HTML Fundamentals

What is HTML for?

HTML defines the structure of a webpage. CSS manages the appearance and layout.

Tags we used last week

  • <h1> through <h6>
  • <p> paragraph
  • <br> line break
  • <b> bold
  • <i> italic
  • <ul> unordered list
  • <li> list item

Attributes

Attributes are added inside the opening tag to give extra information.

Example: The Anchor Tag

The anchor tag creates a hyperlink:

<a href="https://www.example.com">Go to EXAMPLE site</a>

Using CSS in three different ways:

  • Inline CSS
  • Internal CSS
  • External CSS
